System Overview

The Parts Catalog is a web-based application that runs on your local computer or network. Staff can look up automotive parts either by typing keywords or by selecting a vehicle step-by-step. Administrators manage the data through a secure, login-protected dashboard.

🔍

Universal Search

Type any part number, part name, vehicle brand, engine code, or keyword to instantly find matching parts.

🚗

Vehicle Search

Select Brand → Model → Sub Model → Engine to get all parts that fit a specific customer's car.

🛡️

Admin Panel

Add vehicles, add parts, manage existing records, and control user accounts — all with login protection.

💾

Backup & Sync

Download the database to your PC and upload it to another computer to keep all machines in sync.

Starting the Application

🖱️ Double-click start_app.bat

Locate the file called start_app.bat in your Parts Catalog folder and double-click it. A command window will briefly appear to start the server engine in the background.

  • No installation needed each time — just double-click
  • The browser will open automatically to the app
  • A black terminal window shows the network address
  • Keep the terminal window open while using the app

Open your browser and go to http://localhost:PORT (e.g. http://localhost:3042). The browser usually opens automatically.

📱

Access from Other Devices

Other computers or tablets on the same Wi-Fi/LAN network can open the app using the IP address shown in the terminal window. No installation needed on those devices.

⚠️

Keep the Terminal Open

The black command window must stay open while the app is in use. Closing it will shut down the server and the app will stop working.

Search by Vehicle

🚗 Step-by-Step Vehicle Selection

Click the "Search by Vehicle" tab to switch to vehicle-based search. Then select dropdowns one at a time from left to right. Each choice unlocks the next dropdown automatically.

  • 1. Select Brand (e.g. Toyota, Honda)
  • 2. Select Model (e.g. Corolla, Camry)
  • 3. Select Sub Model / Variant (e.g. LE, SE, XSE)
  • 4. Select Engine (optional — any engine if skipped)
  • 5. Select Part Category (e.g. Brake Pads, Oil Filter)
  • 6. Click Find Parts

Understanding Part Types

The system stores two types of parts. Both will appear in search results for the same vehicle.

Feature ✅ Genuine Part 🔶 OEM / Alternative Part
What it is Original manufacturer part Aftermarket equivalent (Bosch, Denso, etc.)
Badge Color Green — Genuine Orange — OEM Brand
Vehicle Link Directly linked to a specific vehicle Linked via compatible Genuine Part Numbers
Engine Link Optional — fits all vehicles with that engine Inherits compatibility from the Genuine part
When both appear Searching for a vehicle or part number returns both Genuine AND all linked OEM alternatives side by side

🔩 Add a New Part

Switch to the Add Part tab. Fill in all fields and click Save Part. After saving, only the Part Number clears so you can quickly add the next part for the same vehicle.

  • Part Type — Genuine or OEM/Alternative
  • Assign to Vehicle — pick from dropdown (Genuine only)
  • Assign to Engine — optional, fits ALL with that engine
  • OEM Brand — only for OEM type (e.g. Bosch)
  • Part Number, Name, Description, Category
  • Specifications — add key/value pairs (e.g. Outer Dia: 68mm)
  • Compatible Genuine #s — for OEM parts, link them to the Genuine part number

Backup & Restore

💾 Backup & Sync Between Computers

All data is stored in a single file called parts.sqlite. Use the Backup tab in the Admin Panel to move this data between computers.

  • Download Database — saves parts.sqlite to your PC
  • Restore / Upload — upload a previously saved file to overwrite the current database
  • ⚠️ Restoring replaces all current data — confirm before proceeding
  • Use this to sync from your shop PC to a laptop, or vice versa
  • Recommended: back up regularly to USB or cloud folder

Tips & Best Practices

📐

Use Specifications Wisely

Add technical specs (e.g. Outer Diameter: 68mm, Thread: M20×1.5) when entering parts. Staff can then search by these values to find interchangeable parts.

🔗

Link OEM to Genuine Parts

When adding an OEM part, always fill in the Compatible Genuine Part Numbers field. This ensures the OEM alternative appears whenever someone searches for the Genuine part number.

🔁

Back Up Regularly

Download a backup of parts.sqlite at the end of each working day or after significant data entry. Store copies on a USB drive or cloud folder.

⚙️

Engine-Based Search

Assign parts to an Engine Type (e.g. 2KD) instead of a specific vehicle if the part fits every car with that engine — regardless of brand, model, or submodel.

🌐

Network Access

Staff on the same Wi-Fi network can access the catalog on tablets or laptops by typing the IP address shown in the terminal (e.g. http://192.168.1.105:3042).

⌨️

Fast Data Entry

After saving a part in the Add Part form, only the Part Number clears. The vehicle, name, and category stay filled so you can quickly add the next variant part.

Frequently Asked Questions

The browser didn't open automatically. What do I do?
Look at the black terminal window that appeared when you ran start_app.bat. Find the line that says "Access on this PC" and copy the address (e.g. http://localhost:3042) into your browser manually.
I searched for a part but got no results. Why?
The part may not yet be in the database. Also check your spelling — the search is case-insensitive but requires at least partial text match. Try searching with fewer characters or a different keyword (e.g. just the brand name).
I entered a part with the wrong details. How do I fix it?
Go to Admin Panel → Manage Parts, find the part, and click Edit. The form will pre-fill with all the existing data. Make your corrections and click Update Part.
Does the system work without an internet connection?
Yes. The application runs entirely on your local computer. As long as the server is started with start_app.bat, the catalog works without any internet connection. Only the font library (Google Fonts) requires internet; without it the UI still functions but uses the default browser font.
How do I move the database to a new computer?
On the old computer, go to Admin Panel → Backup → Download Database to save parts.sqlite. Copy the entire Parts Catalog folder to the new PC, run start_app.bat, go to Admin Panel → Backup → Restore / Upload, and select the downloaded file.
An OEM part doesn't show up when I search the Genuine part number. Why?
When the OEM part was added, the Genuine part number may not have been entered in the Compatible Genuine Part Numbers field. Go to Manage Parts, edit the OEM part, and add the correct Genuine part number in that field.
Can multiple people use the catalog at the same time?
Yes. Any device connected to the same Wi-Fi or LAN network can open the catalog in a browser simultaneously. The server runs on the host computer and handles all requests centrally.
